## üöÄ Getting Started

### Step 1: Login to Admin Panel
1. Open your web browser (Chrome, Firefox, Edge, etc.)
2. Go to: `http://localhost:3000/admin/login`
3. Enter your login details:
   - **Username:** `admin`
   - **Password:** `admin123`
4. Click "Login"

### Step 2: Navigate to Page Content
1. After logging in, you'll see a sidebar on the left
2. Look for **"PAGE CONTENT"** (it might be collapsed - click it to expand)
3. You'll see a list of pages:
   - Home Page
   - About Page
   - Services Page
   - Process Page
   - Plans Page
   - Projects Page
   - Reports Page
   - FAQ Page
   - Contact Page
4. Click on the page you want to edit (e.g., "Home Page")

## ‚úèÔ∏è How to Edit Content

### Understanding the Interface

When you open a page, you'll see:

1. **Help Banner** (Blue box at the top)
   - Shows simple instructions
   - Read this first!

2. **Content Sections** (White boxes)
   - Each box is one piece of content you can edit
   - Numbered for easy reference (1, 2, 3...)
   - Each has a name like "Hero Title" or "About Text"

3. **Edit Fields**
   - Text boxes for short text (titles, headings)
   - Big text boxes for long text (descriptions, paragraphs)
   - Image URL boxes for pictures

4. **Save Buttons**
   - Blue "Save" button next to each section
   - "Save All Changes" button at the top

## üìù Step-by-Step Instructions

### Example: Editing the Home Page Hero Title

1. **Find the Section**
   - Look for the section labeled "Hero Title" (it will be numbered "1")
   - It's in a white box with a blue border

2. **Edit the Text**
   - Click in the text box below "Hero Title"
   - Delete the old text
   - Type your new text (e.g., "Welcome to Our Company")

3. **Save Your Changes**
   - Click the blue "Save" button next to that section
   - Wait for the green success message

4. **See Your Changes**
   - Open your website in a new tab
   - Refresh the page (press F5 or click the refresh button)
   - Your new text should appear!

### Example: Editing a Long Description

1. **Find the Section**
   - Look for sections with "Description" or "Text" in the name
   - These have bigger text boxes

2. **Edit the Content**
   - Click in the big text box
   - You can write multiple paragraphs
   - Press "Enter" to start a new line or paragraph

3. **Save and View**
   - Click "Save"
   - Refresh your website to see changes

### Example: Adding an Image

1. **Find Image Section**
   - Look for sections with "Image" in the name
   - These have special image upload boxes

2. **Add Image URL**
   - Option 1: Paste an image URL in the text box
     - Find an image online
     - Right-click the image
     - Select "Copy image address"
     - Paste it in the box
   - Option 2: Upload an image
     - Click "Click to Upload Image"
     - Select a file from your computer
     - Wait for upload

3. **Save**
   - Click "Save"
   - You'll see a preview of your image

## üí° Tips & Tricks

### ‚úÖ Best Practices

1. **Save Frequently**
   - Save after editing each section
   - Or use "Save All Changes" to save everything at once

2. **Check Your Website**
   - Always refresh your website after saving
   - Use Ctrl+F5 (or Cmd+Shift+R on Mac) for a hard refresh

3. **Write Clearly**
   - Use simple, clear language
   - Break long text into paragraphs
   - Use proper spelling and grammar

4. **Test Images**
   - Make sure image URLs work
   - Use images that are not too large (under 2MB)

### üé® Content Tips

- **Titles:** Keep them short and catchy (under 10 words)
- **Descriptions:** Write 2-3 sentences, break into paragraphs
- **Images:** Use high-quality images, recommended size: 1200x600 pixels

## ‚ùì Common Questions

### Q: I edited text but don't see changes on my website?
**A:** 
1. Make sure you clicked "Save" after editing
2. Refresh your website page (F5 or refresh button)
3. Clear browser cache (Ctrl+F5)

### Q: How do I add a new line or paragraph?
**A:** 
- In big text boxes (textarea), just press "Enter"
- Each time you press Enter, it creates a new line

### Q: Can I undo my changes?
**A:** 
- If you haven't saved yet, just refresh the page
- If you already saved, you'll need to edit it again

### Q: What if I make a mistake?
**A:** 
- Don't worry! Just edit the text again
- Click "Save" to update it
- Your changes are saved immediately

### Q: How do I add an image?
**A:** 
- Find an image online
- Right-click and "Copy image address"
- Paste the URL in the image box
- Click "Save"

### Q: Can I edit multiple sections at once?
**A:** 
- Yes! Edit as many sections as you want
- Then click "Save All Changes" at the top
- This saves everything at once

### Q: What does "Current Content" mean?
**A:** 
- This shows what's currently on your website
- It appears in yellow when you've made changes but haven't saved yet
